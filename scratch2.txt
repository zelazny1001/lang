import Given an intial reference surface of 500 wide and 260 high up which are placed a set of device icons. Let the coordinates of a device on such a surface be dx, and dy. The origin (0,0) is at the top left of the surface. x increases to the right and y increases downward. The requirement is for the 500X260 surface that contains the icons to be correctly placed on the screen of a hardware devoce:
width of screen = sw
height of screen = sh
The goal is to maximize the size of the image displayed on the screen. Since the screen UI is written in angularJS, CSS, and html, one way to do this is to apply this CSS style to the 500X260 background image:
 <div>
     style="
     position: absolute;
     top: 0;
     left: 0;
     height: 100%;
     width: 100%;
     background-size: contain;
     background-position: center center;
     background-repeat: no-repeat;
     ng-style="{'background-image': 'url(' + siteURL + 'php/getImage.php?sid=' + sid + '&id=' + map.media_id + ')'}">
</div>

The goal is to writ a javascript function that takes as inputs dx, dy, sw, and sh, and should return fx and fy. The challenge is to return the value of fx as a percentage of 500, and fy as a percentage of 260. Demonstrate that the returned percentages will correctly map the icon from its original position of dx and dy to the final position on the the output screen.

function calculateFinalPosition(dx, dy, sw, sh) {
  const ow = 500;
  const oh = 260;
  let scale, topOffset = 0, leftOffset = 0;

  const surfaceAspect = ow / oh;
  const screenAspect = sw / sh;

  if (screenAspect > surfaceAspect) {
    // Screen is wider compared to the surface
    scale = sh / oh;
    leftOffset = (sw - ow * scale) / 2;
  } else {
    // Screen is taller compared to the surface
    scale = sw / ow;
    topOffset = (sh - oh * scale) / 2;
  }

  // Calculate final positions
  const fx = ((dx * scale + leftOffset) / sw) * 500;  // As a percentage of 500
  const fy = ((dy * scale + topOffset) / sh) * 260;   // As a percentage of 260

  return {fx, fy};
}

// Example usage
const {fx, fy} = calculateFinalPosition(50, 50, 1024, 768);
console.log(`Final position in terms of original surface: fx = ${fx}, fy = ${fy}`);
