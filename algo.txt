Given 2 java 8 lists (1) List<Chunk> chunkList (2) List<Response> responses
Where 
Class Chunk has method: String getTranscript()
and 
Class Response has method: String getIntent()

Write method:
String utterancesFromPreviousIntent(List<Chunk> chunkList, List<Response> responses)

that checks for the last response in list of responses that has a non-null intent - using the method getIntent()
then
assembles all utterances between the position of the last response to the end of the chunk list
For instance the assembly could be done using    String.join(" ", the_specified_sublist_of_chunkList)

Assume that both responses and chunk list have exactly the same number of elements

==========================================================================

    public static String utterancesFromPreviousIntent(List<Chunk> chunkList, List<Response> responses) {
        int lastNonNullIntentIndex = IntStream.range(0, responses.size())
                .filter(i -> responses.get(i).getIntent() != null)
                .reduce((first, second) -> second)
                .orElse(-1);

        if (lastNonNullIntentIndex == -1) {
            return "";
        }

        return chunkList.subList(lastNonNullIntentIndex + 1, chunkList.size()).stream()
                .map(Chunk::getTranscript)
                .collect(Collectors.joining(" "));
    }
