Given a text passage write a java method "reverseSentences(String input)" that reverses the order of sentences.
For instance, given the input:
"This is the first sentence. And the second. And the third? Here the is fourth"
will return the output:
"Here is the fourth. And the third? And the second. This is the first sentence."

Ensure that every sentence in the output terminates with a period if none is present on the input sentence.

public class SentenceReverser {

    public static String reverseSentences(String input) {
        String[] sentences = input.split("(?<=[.!?])\\s+");
        StringBuilder result = new StringBuilder();

        for (int i = sentences.length - 1; i >= 0; i--) {
            String sentence = sentences[i].trim();
            if (!sentence.endsWith(".") && !sentence.endsWith("?") && !sentence.endsWith("!")) {
                sentence += ".";
            }
            result.append(sentence);
            if (i > 0) {
                result.append(" ");
            }
        }
        return result.toString();
    }

    public static void main(String[] args) {
        String input = "This is the first sentence. And the second. And the third? Here is the fourth";
        System.out.println(reverseSentences(input));
    }
}
