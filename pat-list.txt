1. US Patent 10650812, Issued May 12, 2020 - Deterministic multi-length sliding window protocol for contiguous string entity.
US Patent 10650812 describes a system for extracting verifiable entities from user utterances in an automated calling or interactive voice response (IVR) system. The system includes a receiver for the user's speech, a processor, and a computer-readable medium with executable instructions. These instructions enable the processor to canonicalize the user's speech into tokens (typically words), and then generate a comprehensive series of n-gram sequences from these tokens using a sliding-window protocol. This protocol helps to process various lengths of n-gram sequences to identify and extract verified entities from the utterance efficiently.

The processing involves parallel execution of multiple threads, each handling a different n-gram sequence. The n-gram sequences are processed in descending order, with the system selecting and storing the longest successfully verified n-gram as an entity. If an n-gram is not verifiable, the system retrieves the next n-gram in sequence and repeats the verification process. This method enhances the ability of the IVR system to accurately understand and respond to user requests by extracting key information (entities) from the user's speech with high accuracy, particularly for utterances under thirty tokens in length.

2. US Patent 11164564, Issued June 12, 2020 - Augmented intent and entity extraction using pattern recognition interstitial regular expressions.
US Patent 11164564, Issued June 12, 2020, Augmented intent and entity extraction using pattern recognition interstitial regular expressions
US Patent 11164564 focuses on a dual-module machine learning system for interpreting user utterances, enhancing the accuracy and efficiency of intent and entity extraction in complex linguistic scenarios. The first module employs probabilistic logic to evaluate user utterances and suggest possible intents. When these intents don't clearly exceed a predefined confidence threshold—due to ambiguity or conflict—the system switches to the second module. This module utilizes deterministic logic, particularly regular expression patterns, to more definitively ascertain the user's intent.

This approach addresses the limitations of purely probabilistic methods in handling syntactic variations and similar challenges in natural language processing. By integrating pattern-based analysis, the system can effectively resolve ambiguities that arise in probabilistic analysis, ensuring more precise intent detection. This patent's method is particularly useful in scenarios where distinct utterances bear close syntactic similarities, making traditional machine learning approaches less effective. The system's ability to switch between probabilistic and deterministic logic based on the context and clarity of the user's intent represents a significant advancement in the field of natural language processing and artificial intelligence​.

3. US Patent 10685645, Issued June 16, 2020 - Identification of candidate training utterances from human conversations with an intelligent interactive assistant.
US Patent 10685645 describes a method for creating binary classification models to select candidate training utterances from live utterances. This involves receiving various intents and associated training utterances, creating a binary classification model for each intent that includes a vector representation delineating utterances associated with and disassociated from the intent. Live utterances are received, each assigned an intent, and then transformed into a vector representation. Candidate training utterances are selected based on a comparison between the vector representation of the live utterance and the binary classification model of the assigned intent​.

The patent addresses the need to identify and select candidate training utterances from human conversations using an intelligent interactive assistant. This is crucial for voice response units combined with AI systems, which discern the meaning or intent of human utterances. Traditional AI systems rely on training datasets that need continuous updates to remain effective. The patent aims to automate this process by enabling AI systems to identify live utterances as potential training material, thus surpassing the limitations of human subject matter experts in keeping these systems up-to-date​.

Key aspects of the patent include receiving multiple intents and training utterances associated with each intent, and then combining these into a training utterance collection. From this collection, all n-grams (tokens or unigrams, bigrams, trigrams, and n-grams) are generated, providing a granular and comprehensive linguistic analysis of the utterances. This advanced method of parsing and understanding utterances significantly enhances the AI system's capability to learn and adapt from real-world, live interactions.

4. US Patent 10698500, Issued June 30, 2020 - Touch-based and audio-responsive user interface for elimination of a query's component terms.
Patent 10698500 is an innovative method for enhancing user interaction with voice or text response systems through a touch-based and audio-responsive user interface. This user interface, applicable to devices like smartphones or tablets, is designed to address the common issue of system responses lacking in responsiveness due to reasons such as misspellings, misinterpretations, or incorrect interpretations of homonyms in user queries.

The core of the patent lies in its method of handling user input. Upon receiving input and a system-generated response, if the user finds the response inadequate, they can indicate that it falls below a certain threshold of responsiveness. The system then breaks down the input into component terms (words, phrases, or sentences) and displays them to the user. Users can interactively eliminate terms that they believe led to the misunderstanding, either by touching them on the interface or through other means, to refine their query. This elimination process is designed to enhance the system's responsiveness to user queries and reduce misunderstandings, thereby improving the overall user experience with the interface​.

5. US Patent 10825448, Issued November 3, 2020 - Quantum technology for use with extracting intents from linguistics.
US patent 10825448 is a novel approach to understanding human utterances in interactions with machines. The patent describes an apparatus for mapping a user's utterance onto various intents. This apparatus includes a processor and an intent training database containing tokens and intents. The processor uses these to generate a series of matrices: a token-intent matrix, a token-cognitive matrix, a cognitive-comprehension matrix, and an intent-cognitive matrix. The cognitive-comprehension matrix is key, acting as a space of entanglement between the token-cognitive matrix and the intent-cognitive matrix, allowing for a deeper understanding of the relationship between utterances and intents.

Each utterance, along with its equivalent forms, maps to a specific intent, excluding others. This mapping signifies the quantization of utterances from their broader domain to specific meanings. Interestingly, even minor variations in an utterance can alter its entire meaning, demonstrating that the relationship between utterances and intents is complex and not solely dependent on syntax. The patent introduces a method to define this relationship, referred to as the cognitive-comprehension space, which is represented using a matrix that can be represented using a matrix that can be singlular-value decomposed.

7. US Patent 11070377, Issued on July 20, 2021 - Blended virtual machine approach for flexible production delivery of intelligent business workflow rules.
Patent 11070377 introduces a method and system for the blended virtual machine approach for flexible production delivery of intelligent business workflow rules. This patent primarily focuses on software delivery, particularly on deploying software obtained from multiple sources.

The core concept involves blending uncompiled source files and compiled binaries within a single Runtime container. This process encompasses several steps such as receiving data at a configuration database, invoking a scheduler, determining the need to rerun software code, performing hash functions on source files, and handling updated source files. The method aims to effectively store, parse, and load these files, ultimately compiling binaries and implementing their associated logic.

The patent addresses a common challenge in software development, where business rules, typically encapsulated in workflows, frequently change. These changes often occur faster than the typical release schedules for compiled binary files, which are usually set at intervals of eight to twelve weeks. The need for a system that can maintain a standard release schedule while accommodating rapid, on-demand updates of business rules is emphasized.

10. US Patent Pending 20200394469, published December 17, 2020, Systems and methods for automated degradation-resistant tuning of machine-learning language processing models
Patent 20200394469 introduces methods and systems for the automated tuning of machine-learning (ML) models used in language processing. These systems and methods are designed to handle inoperative input utterances by identifying them, retrieving their associated feature sets, and generating updated utterance-feature-intent (UFI) mappings. The ML model is then retrained with this updated UFI mapping, and its accuracy is compared before and after retraining. If the system's accuracy improves, the new UFI mapping is deployed; if not, the UFI mapping is further amplified for better results​​.

The patent application primarily relates to artificial intelligence (AI) systems, particularly those that employ ML models for language processing. It highlights the importance of such systems in interpreting and responding to customer queries, emphasizing the direct impact of their success or failure on customer satisfaction. The patent acknowledges the challenge of maintaining accuracy across the entire ML system, especially when updates might improve accuracy for some utterances but degrade it for others. Thus, it emphasizes the need for an automated ML tuning system that not only improves accuracy but also resists degradation in other parts of the system​​.

The system includes an ML model trained on a database-stored training set comprising various utterances and intents, each assigned a set of features. The tuning module is a crucial component, designed to enhance the comprehension of the ML model while preventing degradation. It flags inoperative utterances (those the system fails to interpret accurately), retrieves related feature sets, and creates UFI mappings based on these sets. The model is updated with this mapping, and its performance is evaluated before and after the update to ensure overall system improvement
 
11. US Patent Pending 20230222294, Published July 13, 2023, Anaphoric reference resolution using natural language processing and machine learning
Patent 20230222294 is focused on anaphoric reference resolution using natural language processing (NLP) and machine learning. Anaphoric references are implicit entity references. The purpose of this innovation is how earlier explicit entity references are connected to later explicit references. The invention concerns a device designed to handle sentences containing both explicit and implicit references. The device inputs words into a machine learning model to generate a feature vector, identifies implicit references, and finds matching explicit references. It then replaces the implicit references with the corresponding explicit references in the sentence, enhancing the clarity and understanding of the text​.

The patent addresses a significant challenge in natural language processing: the difficulty of computer systems to accurately process messages containing implicit references, like "it," "they," and "this." Such implicit references often refer to items previously mentioned, and without proper resolution, they can lead to misunderstandings and inefficient use of processing resources​.

This system presents several practical applications, offering a technical solution that modifies messages by replacing implicit references with explicit ones before forwarding to a network device for further processing. This approach not only improves the interpretation of messages by the network device but also enhances the overall efficiency and throughput of the information system, making it a significant advancement in the field of natural language processing and machine learning​.
